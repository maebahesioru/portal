# uno — ブラウザ版UNO

unoは、ブラウザ上でAI相手にUNOをプレイできるカードゲームです。独自のハウスルールシステムと3種類のオプションワイルドカードを搭載しています。

## ゲームシステム

- **AI対戦**: 複数のAI相手と対戦（aiPlayer.ts）
- **ハウスルール**: 山札重ね、ワイルド+4いつでも、複数枚同時出し、連番出し、アクションカード上がり禁止、ラストマンスタンディング
- **オプションワイルドカード**: Swap（手札交換）、Shuffle（シャッフル）、Custom（カスタム）
- **ワイルド+4チャレンジ**: 正式ルールのチャレンジ判定
- **スコアリング**: 日本式/国際式切り替え
- **グリーン/ピンクグラデーション**: ダークテーマUI

## 技術スタック

- Next.js 14.2、React 18.3、TypeScript、Tailwind CSS 3.4
- GameSetup → GameBoard フロー（クライアントコンポーネント）
- 純粋なフロントエンドゲーム（外部API・DBなし）
- PostCSS\n
## 開発の裏話

AIプレイヤー（aiPlayer.ts）は状態ベース戦略で実装。ワイルド+4チャレンジ判定やハウスルールのオンオフをすべてクライアントサイド完結させるのに苦労しました。
