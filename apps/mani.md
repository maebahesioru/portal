# mani — ヒカマニ語録翻訳サービス

mani（mani!?翻訳）は、入力した日本語テキストをAIがヒカマニ界隈の語録・方言に変換するスタイル翻訳サービスです。通常の日本語をヒカマニ風の言い回しに書き換え、使用した語録の解説も表示します。

## 主な機能

- **ヒカマニ語録翻訳**: 日本語→ヒカマニ語録にAIが変換
- **複数翻訳モード**: ヒカマニ / コウヘイ / カツヒコ / 淫夢 / 日本語の5モード
- **ストリーミング表示**: SSEで翻訳結果をタイプライター風に逐次表示
- **推論過程の可視化**: AIの思考プロセスも表示
- **画像翻訳対応**: 画像を添付して翻訳可能
- **翻訳履歴**: 過去の翻訳結果をlocalStorageに保存
- **翻訳スタイル調整**: natural/literalの翻訳スタイル選択

## 技術スタック

- Next.js 16.1.1、React 19.2、TypeScript、Tailwind CSS 4
- Gemini AI（Google Generative AI）
- react-markdown + remark-breaks
- OpenGraph画像生成（opengraph-image）

## 開発の裏話

約80項目のヒカマニ語録コーパスを手動収集し、システムプロンプトに埋め込んでいます。ゲイマスオ・淫夢など複数の界隈方言に対応するため、モードごとに異なるプロンプトテンプレートを用意しています。
